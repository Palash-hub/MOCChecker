# MOCChecker
MOCChecker — это легковесная и быстрая CLI-утилита для проверки целостности связей в вашей базе знаний Obsidian. Инструмент идеально подходит для тех, кто строит "Personal Wikipedia" на основе метода MOC (Map of Content).
======
### Как запустить проект у себя?
#### Клонируйте репозиторий:
`git clone https://github.com/ТВОЙ_ЛОГИН/MOCChecker.git`
#### Перейдите в папку проекта и соберите его:
`dotnet build --configuration Release`
#### Запустите тесты, чтобы убедиться, что всё работает:
`dotnet test`
======
### Запуск через CLI
#### Основной запуск
`MOCChecker --path "<VAULT_PATH>"`
#### С увеличенным количеством потоков для проверки внешних ссылок
`MOCChecker --path "<VAULT_PATH>" --threads <THREADS_COUNT>`
#### Игнорировать проверку вложенных изображений
`MOCChecker --path "<VAULT_PATH>" --ignore-images`
======
### Флаги
Флаг|Описание
 --- |--- 
**-p, --path** |(Обязательно) Полный путь к папке с вашими заметками
**-i, --ignore-images** |Пропустить валидацию файлов изображений (.png, .jpg и др.)
**-t, --threads** |Лимит одновременных HTTP-запросов (по умолчанию 10)
